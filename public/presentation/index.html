What is TelcoVantage ERP?

A full-stack enterprise operations platform purpose-built for the telecom industry in the Philippines.

  • Replaces spreadsheets & disconnected tools
  • Provides a single, unified, auditable system
  • Powered by Google Gemini AI for deep insights
  • Enforces rigorous compliance and financial rules

1. Getting Started

Login, Navigation, and Basic Controls

Login & Security

  • Access: Login via email and password at /login.
  • Security: All routes are protected by role-based access control.
  • Profile Management: Manage your password and avatar from the Profile page.

Navigation Essentials

  • Sidebar: Collapsible menu containing all system modules.
  • Global Search (⌘K): Instantly search vendors, POs, projects, and more across the entire system.
  • Notification Bell: Real-time alerts for POs, invoices, and expiring documents.
  • Theme Toggle: Switch between Light, Dark, and Midnight modes.

2. Command Center

Your Real-time Operational Pulse

Key Metrics (KPIs)

The dashboard surfaces critical numbers instantly:

  • Current Liability: Total unpaid invoice amounts.
  • Active POs: Open financial commitments.
  • Pending Vendors: Vendors awaiting accreditation.
  • Expiring Docs: Critical renewals needed in the next 30 days.

System Activity Feed

  • A real-time audit log of everything happening in the system.
  • Color-coded actions: CREATE (Green), UPDATE (Blue), DELETE (Red).
  • Fully traceable—know exactly who did what and when.

3. Vendor Management

Full lifecycle. Zero surprises.

Vendor Directory

  • Completeness Indicator: Green/red dots show if a profile is missing required fields (TIN, Bank, etc.).
  • Accreditation Progress: Visual bar showing X/14 required documents approved.
  • Multi-currency: Supports both PHP and USD vendors.

Vendor Lifecycle

Pending → Active → Inactive

  • New vendors start as Pending.
  • Must be activated by an Admin or Procurement officer.
  • Only Active vendors can be issued new Purchase Orders.

4. Compliance Hub

The 14-Point Accreditation System

The Accreditation Matrix

A bird's-eye view of vendor compliance across 14 document types (NDA, SEC, PCAB, BIR, etc.).

  • ✅ Approved: Valid and verified.
  • 🕐 Pending Review: Uploaded but awaiting admin approval.
  • ⚠️ Expired: Document validity period has passed.
  • ❌ Missing: Document has not been submitted.

5. Purchase Orders

Governed, trackable commitments.

PO Creation & Guardrails

  • Auto-numbering: Sequential `PO-YYYY-XXXX` generation.
  • NDA Gate: System blocks PO creation if the vendor does not have a signed, valid NDA.
  • Details: Build POs with specific line items and deployment site details (Region, Nodes, Cable Length).

PO Status Workflow

Draft → Issued → Partially Paid → Paid

  • Financial Dashboard: Every PO page features a balance ring showing % paid vs remaining.
  • Export: Download POs as PDF or DOCX.
  • In-Browser Editor: Edit the DOCX version directly within the ERP.

6. Invoicing & Payments

Know where every peso is.

Recording Invoices

  • PO Amount Guard: The system prevents invoice amounts from exceeding the remaining PO capacity.
  • Links directly to a specific PO.
  • Overpayment detection triggers red warning banners if limits are overridden.

Recording Payments

The Auto-Cascade Logic:

  1. Record a payment (Full, Installment, Down Payment).
  2. System updates Invoice status (e.g., to Paid).
  3. System automatically recalculates the linked PO.
  4. PO status automatically updates (e.g., to Partially Paid or Paid).

7. Projects, Contracts & CRM

Projects & Contracts

  • Projects: Link multiple vendors and POs to a single internal project.
  • Safety Guards: Cannot unlink a vendor if they have open POs on the project.
  • Master Contract Repository: Track contract values, start/end dates, and view embedded PDFs.

CRM (Customer Management)

  • Manage Customer accounts and primary/secondary contacts.
  • Opportunity Pipeline: Track prospective projects, estimated values, and follow-up dates.
  • Upload and version customer documents.

8. Operational Vaults

Documents, HR, and Assets

Documents & Assets

  • Company Library: Centralized folders for Legal, HR, Finance, and Templates.
  • In-App Preview: View PDFs and Office docs without downloading.
  • Asset Registry: Track laptops, equipment, serial numbers.
  • Depreciation Calculator: Built-in straight-line depreciation tracking.

HR & Employee 201 Vault

  • Manage employee profiles, roles, and employment status.
  • 201 Vault: Securely store employee documents (IDs, contracts).
  • Only Admins can invite new users and assign roles (Finance, Procurement, etc.).

9. Advanced Features

AI Assistant & Vendor Portal

Gemini AI Assistant

Your intelligent co-pilot, accessible via the floating chat bubble.

  • Ask Anything: "Who are our top vendors?", "List pending POs."
  • Document Analysis: Upload a PDF and ask the AI to summarize obligations or extract expiry dates.
  • Actions: AI can create vendors or run imports for you (requires your confirmation).

The Vendor Portal

  • Magic Links: Generate secure, 7-day expiring links to send to vendors.
  • Self-Service: Vendors can upload their own accreditation documents.
  • E-Signature: Vendors can draw their signature directly on screen; the system auto-stamps it onto the PDF with IP and timestamp.
  • AI OCR: The system auto-extracts expiry dates from vendor uploads.
